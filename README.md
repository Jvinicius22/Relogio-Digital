Relógio Digital com Calendário

Um Relógio Digital com Calendário desenvolvido utilizando HTML, CSS e JavaScript.
O projeto exibe horas, minutos e segundos em tempo real, além de mostrar informações completas da data atual.

O projeto apresenta:
  Relógio digital em tempo real
  Dia da semana ou mês
  Mês
  Ano
  Data completa formatada
 Interface simples e minimalista, limpa para melhor entendimento
 
 Este projeto foi desenvolvido utilizando:
  HTML
  JavaScript
  CSS
